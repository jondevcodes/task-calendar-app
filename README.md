# 🧠 Task Calendar App (CLI)

A clean, object-oriented task tracker built in Python.

## ✅ Features
- Add tasks with due dates
- View tasks sorted by date
- Mark tasks as complete ✅
- Delete tasks 🗑️
- Persistent storage using JSON

## 🛠️ Technologies
- Python 3
- OOP (classes, methods)
- CLI interface
- JSON for file persistence
